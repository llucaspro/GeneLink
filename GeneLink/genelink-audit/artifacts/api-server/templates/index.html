Construído sobre Dados Científicos Reais

O GeneLink consulta a API Entrez do NCBI (eutils.ncbi.nlm.nih.gov) em tempo real para cada busca de genes. Não há resultados simulados, dados fictícios ou informações científicas fabricadas. Se um gene não existe no NCBI, não aparecerá aqui.
